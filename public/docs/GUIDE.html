Daily Workflow
  1. Open Command Center for run status and priorities.
  2. Launch Chat and run your mission with Super Agent.
  3. Switch to Lux AI or Lux Agent for code-intensive steps.
  4. Use approvals for high-risk actions.
  5. Capture reusable outcomes as playbooks.

Example mission prompt:

Plan and execute a launch checklist for my product, create tasks, run step one, and report blockers with options.
Memory, Safety, and Backups

Memory + Learning

Backups + Restore

  • Keep scheduled backups for fast recovery.
