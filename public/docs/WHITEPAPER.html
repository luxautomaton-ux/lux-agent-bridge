The Unified Private Agent Platform

Lux AI is an enterprise-grade AI agent platform designed for private, local-first development. Operates entirely on your hardware with zero API keys, zero data exfiltration, and complete user control.

Deployment Options

Local Installation

git clone https://github.com/luxautomaton-ux/lux-agent-bridge.git cd lux-agent-bridge npm install npm start # Access: http://localhost:8787

USB Portable

  • Copy folder to USB drive
  • Run Start-Lux-AI.bat (Win)
  • Run Start-Lux-AI.command (Mac)
  • Browser opens automatically
