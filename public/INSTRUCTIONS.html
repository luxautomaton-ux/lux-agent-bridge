📖 Lux Agent Studio - Complete Instructions

Your private AI agent orchestration platform

🚀 Quick Start Guide
1

Start the Server

Navigate to the project folder and run: cd /Users/asaspade/Desktop/lux-agent-bridge && npm start

2

Access the Interface

Open your browser to http://localhost:8787

3

Configure OpenMono Agent

Go to Tools → Skills to set your OpenMonoAgent path in environment variables (OPENMANUS_DIR)

4

Start Your First Mission

Go to Chat and describe what you want to accomplish. The agent will orchestrate the workflow.

💬 Using Chat

Access via: Control → Chat

  1. Select Agent (OpenMono or Manus)
  2. Choose Mode (Assisted/Autonomous/Manual)
  3. Type your mission or use Tools dropdown
  4. Press Send or Ctrl+Enter
🖼️ Photos & Media

Access via: Chat → Tools → Upload Files

  1. Click Tools → Upload Files
  2. Drag & drop or select images/PDFs
  3. The agent analyzes automatically
  4. View results in chat
📊 Reports & Slides

Generate via Chat by asking:

"Create a report on [topic]"
"Generate slides about [subject]"
"Summarize the data from [file]"
🔬 Research Mode

Use research tool in chat:

Click Tools → Research Topic
Enter topic, select depth
Click Start Research
🌐 Web & Browser

Search web via:

Click Tools → Web Search
Enter query, view results
Direct links to GitHub, docs
📱 App Creation

Build apps through chat:

"Create a React app for [purpose]"
"Build a Python CLI tool that [does X]"
"Make a web app with [features]"
⚙️ Configuration

Environment variables to set in .env:

PORT=8787 OPENMANUS_DIR=/path/to/OpenManus LUX_API_TOKEN=your-token-here ALLOWED_PROJECT_ROOTS=/path/to/projects REQUIRE_APPROVAL_AUTONOMOUS=true
