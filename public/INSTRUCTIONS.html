Quick Start

Use this exact path the first time:

1

Start the app

Run Lux AI Studio from the project root.

npm start
2

Open the workspace

In browser, open local UI and verify health is green.

http://localhost:8787
3

Choose your default agent

In Chat, set Super Agent as your default for mixed research + execution tasks.

4

Run your first mission

Example prompt:

Create a 5-step launch plan for my app, then generate tasks and start with step 1.
Agent Selection Cheat Sheet

Super Agent

Best default. Use for planning + web research + execution in one flow.

Lux AI (Native)

Use for structured coding and local project actions with tighter control.

Lux Agent

Use for deep refactors, tests, and code-heavy implementation missions.

Security Guard

Use for vulnerability checks and approval-gated operations.

Section-by-Section Flow

Chat

  • Pick agent and mode
  • Attach files or research topics
  • Approve risky operations when prompted

Task Queue

  • Track active missions
  • Inspect run logs
  • Retry or branch failed runs

Skills

  • Sync Awesome Skills
  • Verify link health
  • Remove broken entries
